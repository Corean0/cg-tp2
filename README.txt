# Lego Park

# Instruções

	Bibliotecas Utilizadas:
	Glew, SOIL, FreeGlut, SDL e SDL-mixer,

	Instalação das bibliotecas utilizadas:
	sudo apt install libsoil-dev libsdl1.2-dev libglew-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-ttf2.0-dev freeglut3-dev

	Compilação e Execução:
	Abrir o terminal no diretório "Makefile"
	make       -> Compila o programa
	make run   -> Compila e executa o programa
	make clean -> Apaga arquivos temporários

# Comandos

	**w/s/a/d**   -> Movimentação 
	**Mouse**     -> Movimentação da Camera
	**n**         -> Nevoa 
	**l**         -> Iluminação
	**1/2/3**     -> Alternar Cameras
	**ESC**       -> Exit
